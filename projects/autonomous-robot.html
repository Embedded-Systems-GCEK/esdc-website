Humanoid Robot

AI Assistant Robot With Legs , Ears and Mouth

Project Overview

The Humanoid Robot project focuses on developing an intelligent robotic system capable of navigating complex environments independently. The robot comes with assistance capabilities and advanced obstacle detection and avoidance mechanisms.

Key Features

  • Assistance Feature built with AI
  • Data logging and analysis
  • Tracking User Communications and Intelligent storing and Analyzing
  • Real-time path planning and navigation
  • Motor control with speed regulation
  • Wireless communication capabilities
  • Data logging and analysis

Technologies Used

ESP32 Lidar Raspberry Pi 5 Edge AI Ollama Gemini Python Flask REST APIs Swagger PID Control Inverse Kinematics

Project Goals

Our goal is to create an Humanoid Robot with AI capabilities. And to create a robust autonomous navigation system that can adapt to various environmental conditions. The robot should be able to follow predetermined paths while making intelligent decisions to avoid obstacles and optimize its route.
